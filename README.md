# Heart-Disease-Detection-Web-App
A Flask web application for heart disease detection using a stacked ensemble machine learning model (Naive Bayes + Random Forest). Interactive interface for healthcare risk assessment.

# Heart Disease Detection System

This repository contains a **Flask web application** that allows users to interactively detect heart disease using a **stacked ensemble machine learning model** combining **Naive Bayes and Random Forest**. 

## Features

- End-to-end **ML pipeline** integrated into a web app
- User-friendly form to input clinical parameters
- **Real-time predictions** of heart disease status
- Evaluated using **Accuracy, Precision, Recall, and ROC-AUC**
- Built for **healthcare analytics** and portfolio demonstration

## Getting Started

1. Clone the repository:
2. pip install -r requirements.txt
3. python app.py
4. Open your browser at http://127.0.0.1:5000/ to start detection.
